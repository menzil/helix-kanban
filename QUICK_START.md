# 快速开始

## 🚀 运行应用

```bash
cargo run --release
# 或
./run.sh
```

## ⌨️ 快捷键速查

### Project List（项目列表）

| 按键 | 功能 |
|------|------|
| `j` / `↓` | 向下移动 |
| `k` / `↑` | 向上移动 |
| `Enter` | 打开项目 |
| `n` | 新建项目 |
| `e` | 编辑项目 |
| `q` / `ESC` | 退出应用 |

### Board View（看板视图）

| 按键 | 功能 |
|------|------|
| `h` / `←` | 切换到左边的列 ✨ |
| `l` / `→` | 切换到右边的列 ✨ |
| `j` / `↓` | 在当前列中向下移动 |
| `k` / `↑` | 在当前列中向上移动 |
| `a` | 新建任务 ✨ |
| `e` | 编辑任务 ✨ |
| `d` | 删除任务 ✨ |
| `H` (Shift+h) | 将任务移动到左边的列 ✨ |
| `L` (Shift+l) | 将任务移动到右边的列 ✨ |
| `q` / `ESC` | 返回项目列表 |

✨ = 本次修复的功能

### 输入对话框

| 按键 | 功能 |
|------|------|
| `字母/数字/符号` | 输入内容 |
| `Backspace` | 删除字符 |
| `Enter` | 确认 |
| `ESC` | 取消 |

## 📝 快速测试

1. **启动应用** → 看到项目列表
2. **按 `Enter`** → 进入 Board 视图
3. **按 `h` 和 `l`** → 测试列切换（应该能看到列的高亮切换）
4. **按 `a`** → 创建新任务（输入任务名，按 Enter）
5. **选中任务，按 `L`** → 将任务移动到下一列
6. **选中任务，按 `e`** → 编辑任务
7. **选中任务，按 `d`** → 删除任务

如果以上操作都正常工作，说明快捷键修复成功！✅
